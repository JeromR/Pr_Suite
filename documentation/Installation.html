Installation

Pr_Suite Introduction ››

  • Download Pr_Suite from http://www.parimalvfx.com/rnd/pr_suite/


  • After downloading extract the Pr_Suite package.


  • Copy Pr_Suite package to one of the following directories based on your operating system:


Operating System

Installation Directory

Windows

C:\Users\username\.nuke

Mac

Users/username/.nuke

Linux

/home/username/.nuke


  • If you are installing Pr_Suite in custom NUKE_PATH make sure to copy the whole Pr_Suite package as it is.


  • After copying the package add following lines to your root init.py, if you don't have init.py create one by copy and pasting following lines in a text editor and save it as init.py in your .nuke directory:


nuke.pluginAddPath( "./Pr_Suite" )


  • Now restart Nuke and you can see Pr_Suite is successfully installed if Pr_Suite icon is visible menu bar and Pr_Suite menu is visible in Nuke menu (Pr_Suite Nuke menu will have icon for Windows and text for Mac).

  • Another way to check if Pr_Suite is successfully installed is to create a Pr_ShuffleMatte and see if color buttons are properly visible.

 Installation wrong


If installation is wrong, make sure you have copied the whole package as it is and you have not modified the menu.py and init.py included in the package.
