Known Bugs and Workarounds:

  • Pr_RGBLuma v1.0:

      Changing options in 'Adjust in' pulldown choice for the first time, intensity knob will get a 'nan' value.

To overcome this, change to desired option from 'Adjust in' pulldown choice and then set intensity knob to zero before adjusting it. After this you change options from 'Adjust in' pulldown choice without any 'nan' value problem in intensity knob.
